# vue-qbao 保险前端页面-长险

## 安装

```js
框架: Vue 3 + TypeScript + Vite
安装: npm install
vscode插件： Vetur、Vue Language Features、Vue 3 Snippets
```

#### Eslint setting 设置

```json
// eslint fix 保存
"editor.codeActionsOnSave": {
    "source.fixAll.eslint": true,
    "source.fixAll.stylelint": true
},
// prettier 格式化
"prettier.semi": false,
"prettier.singleQuote": true,
```
